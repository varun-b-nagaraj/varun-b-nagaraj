# About Me

Software engineer and robotics developer focused on building scalable systems across AI, simulation, automation, and developer infrastructure.

My work combines full-stack engineering, robotics, and applied AI with an emphasis on performance, usability, and real-world impact. I enjoy building systems that are both technically ambitious and practical to use.

Currently working on robotics simulation platforms, AI-native workflow systems, autonomous robotics software, and quantum computing research.
